# Automatización y Control

## Descripción

Repositorio orientado al desarrollo de proyectos de automatización, instrumentación y control utilizando microcontroladores, sensores y actuadores.

El objetivo de este repositorio es demostrar conocimientos prácticos en monitoreo de variables físicas, adquisición de datos, control automático y automatización de procesos, aplicando conceptos utilizados en entornos industriales, sistemas embebidos e Industria 4.0.

Los proyectos fueron desarrollados utilizando plataformas de prototipado y simulación, permitiendo validar soluciones reales de control y supervisión sin requerir hardware físico.

---

## Objetivos de Aprendizaje

* Implementar sistemas de monitoreo utilizando sensores.
* Desarrollar soluciones de automatización basadas en microcontroladores.
* Aplicar técnicas de control ON/OFF.
* Integrar actuadores en sistemas embebidos.
* Diseñar sistemas de supervisión de variables físicas.
* Simular escenarios utilizados en la industria y la automatización.

## Proyectos Incluidos

### 01 - Monitoreo Inteligente de Temperatura

Sistema de monitoreo ambiental basado en ESP32, sensor DHT22 y pantalla OLED SSD1306 para visualizar temperatura y humedad en tiempo real.

**Conceptos aplicados:**

* Instrumentación electrónica
* Monitoreo ambiental
* Comunicación I2C
* Sistemas embebidos

---

### 02 - Control Automático de Ventilación

Sistema de control automático que activa un ventilador cuando la temperatura supera un umbral configurado.

**Conceptos aplicados:**

* Control ON/OFF
* Automatización industrial
* Supervisión de variables
* Control de actuadores

---

### 03 - Control de Nivel de Tanque *(Próximamente)*

Sistema de monitoreo de nivel utilizando sensores ultrasónicos para supervisar el nivel de líquidos en depósitos.

**Conceptos aplicados:**

* Instrumentación industrial
* Sensores ultrasónicos
* Medición de nivel
* Automatización de procesos

---

### 04 - Control Automático de Iluminación *(Próximamente)*

Sistema que regula automáticamente la iluminación según las condiciones de luz ambiente.

**Conceptos aplicados:**

* Sensores analógicos
* Automatización residencial
* Eficiencia energética
* Domótica

---

### 05 - Sistema de Riego Inteligente *(Próximamente)*

Sistema automatizado que activa el riego en función de la humedad del suelo.

**Conceptos aplicados:**

* Agricultura inteligente
* Automatización agrícola
* Control automático
* Sensores de humedad
